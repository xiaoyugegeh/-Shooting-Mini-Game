# 贡献指南

感谢你对 VALORANT TRAINING GROUND 项目的关注！

## 如何贡献

### 1. 提交 Issue

如果你发现了 bug、有功能建议，或者对游戏有任何想法，欢迎提交 Issue。

提交 Issue 时，请尽量包含以下信息：
- 问题描述
- 复现步骤
- 浏览器版本
- 截图（如果适用）

### 2. 提交 Pull Request

1. Fork 本仓库
2. 创建你的特性分支：`git checkout -b feature/my-feature`
3. 提交你的改动：`git commit -m 'feat: add some feature'`
4. 推送到分支：`git push origin feature/my-feature`
5. 提交 Pull Request

### 3. 代码风格

- 使用 ES Modules 模块化开发
- 代码注释使用中文
- 保持简洁，避免过度工程化
- 性能敏感代码避免每帧创建新对象

### 4. 提交信息规范

提交信息请使用以下前缀：

- `feat:` 新功能
- `fix:` 修复 bug
- `docs:` 文档更新
- `style:` 代码格式调整（不影响功能）
- `refactor:` 重构
- `perf:` 性能优化
- `test:` 测试相关
- `chore:` 构建/工具相关

## 社区行为准则

- 友善交流，尊重每一位贡献者
- 保持讨论围绕项目本身
- 欢迎新手提问

再次感谢你的贡献！
